# Goal

## 正式 Objective

为初学者构建一本由浅入深、带可靠 references 的 AI Agent 中文学习手册内容蓝图。系统调查 AI Agent 领域的核心主题和高质量资料，按广度、深度、精度、正确性四个标准进行校验和整理，产出主题地图、章节目录、资料可信度规则、资料卡片模板、章节写作模板、推荐来源清单、references 组织规范和待验证问题清单，为后续生成 GitHub Pages 电子书打基础。

## 第一阶段范围

第一阶段不追求完整书稿，也不优先做页面设计。重点是建立内容治理体系。

完成标准：

- 有完整主题地图。
- 有由浅入深的章节结构。
- 有资料可信度分级标准。
- 有资料卡片模板。
- 有章节写作模板。
- 有 references 标注规范。
- 有不少于 30 个候选高质量资料来源。
- 每个核心主题至少有 2-3 个高可信 references 候选。
- 明确哪些内容适合初学者，哪些属于进阶阅读。
- 明确哪些结论已验证，哪些还需要交叉验证或实验验证。

## 非目标

- 不做未校验链接合集。
- 不直接复制网络文章。
- 不把营销材料当作事实来源。
- 不为了显得高级而堆术语。
- 不在没有 references 的情况下写确定性结论。
