## 已准备的真实验证入口

脚本已准备一个最小 RAG citation synthesis 流程：

- 使用标准库关键词 overlap 从固定小语料中选择 chunk。
- 请求真实 Responses API 生成结构化 JSON：`answer`、`grounded`、`citations` 和 `missing_evidence`。
- 校验 `grounded=true` 时 citation 是否只引用检索到的 `chunk_id`。
- 校验无证据问题是否返回 `grounded=false`，且不编造 citation。
- 对答案做最小语义检查，记录失败原因。
