## 已准备的真实验证入口

脚本已准备一个最小 Responses API production observability 流程：

- 运行 1-5 次小请求，默认 3 次。
- 记录 request count、HTTP status、usage tokens 和 response id。
- 尝试读取 request / token rate-limit headers。
- 计算平均 latency 和 P95 latency。
- 在提供价格环境变量时估算成本；未提供价格时明确标记为 unknown。
- 根据 budget threshold 输出 `continue`、`degrade` 或 `unknown_without_prices`。
