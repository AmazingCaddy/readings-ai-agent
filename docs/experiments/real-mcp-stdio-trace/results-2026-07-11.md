- 日期：2026-07-11

## 已验证的本地行为

- Host 启动一个 stdio server 进程，并通过 JSON-RPC 发送 `initialize`、`tools/list`、`tools/call`、`resources/list` 和 `resources/read`。
- Server 返回 tools catalog、resources catalog、tool result 和 resource contents。
- Host 在只读工具调用前记录 approval，并允许请求转发。
- Host 在写工具调用前记录 approval rejection，未把该写工具请求转发为实际写入。
- Host 在 resource read 后记录普通资源和不可信注入候选资源的 review decision。
- Trace 中包含 17 个事件；示例中的 `secret=example-token` 被脱敏，本次输出 `leaked_secret_in_trace=false`。

## 结论状态

- 可支撑：MCP host / client / server 进程边界、stdio JSON-RPC 消息流、tool catalog、resource catalog、host approval、resource review 和 trace redaction 的最小本地 harness 已完成。
- 不能支撑：本结果不证明官方 MCP SDK、真实 host UI、OAuth、URL mode elicitation、sampling、真实文件 sandbox、token audience 校验、remote MCP connector、tunnel 或跨 host 默认行为。
