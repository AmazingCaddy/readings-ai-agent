## 一句话总结

Generative Agents 是理解“记忆、反思、规划”如何组合成 believable agent behavior 的重要论文。

## 核心结论

- 论文摘要描述了一个把 LLM 扩展为可存储经验、综合高层反思并动态检索记忆来规划行为的 agent 架构。
- 摘要提到 observation、planning 和 reflection 组件分别对行为可信度有关键贡献。
