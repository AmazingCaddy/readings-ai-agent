## 使用规则

- 只有完成链接复核、精读和交叉验证的结论，才能进入章节正文。
- 只完成链接复核的资料可以保留为候选 reference，但不能支撑确定性结论。
- 工程建议优先要求官方文档、源码示例和失败模式三者同时覆盖。
