# 资料库

资料进入正文前必须先完成 source card。

## 流程

1. 在 `seed-candidates.md` 中登记候选资料。
2. 为资料创建 source card。
3. 完成来源复核。
4. 和至少 1-2 个独立来源交叉验证关键结论。
5. 必要时设计实验。
6. 决定是否进入章节正文。

## Source Card 位置

```text
docs/sources/source-cards/
```

命名建议：

```text
YYYY-source-slug.md
```

示例：

```text
2022-react-paper.md
2024-mcp-spec.md
2025-openai-agents-sdk.md
```
